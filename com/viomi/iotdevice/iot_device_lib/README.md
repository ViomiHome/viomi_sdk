# 模拟小米WIFI模组的串口SDK
# 使用方法
## 一、基础配置使用
### 1.在你项目根目录下的 build.gradle 文件配置 maven 仓库地址：

    repositories {
         maven { url "https://raw.githubusercontent.com/ViomiHome/viomi_sdk/master" }
    }
  
### 2.然后在 app/build.gradle 文件中添加依赖:
    dependencies { 
        implementation "com.viomi.iotdevice:iot_device_lib:1.0.0"
    }
